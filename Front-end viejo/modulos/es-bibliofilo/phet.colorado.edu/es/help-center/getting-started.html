¿De donde proviene el nombre "PhET"?

Originalmente nuestro proyecto se enfocaba en las simulaciones de física y por lo tanto su nombre en inglés era *Ph*ysics *E*ducation *T*echnology project, o *PhET*. Cuando nos expandimos hacia la química, biología, matemáticas y otras áreas, decidimos mantener el nombre de *PhET*, pero ya no lo consideramos un acrónimo.

¿Cómo puedo incrustar las simulaciones de PhET?

Haga clic en el botón "Embed" debajo de la simulación que quiere incrustar. Esto creará un cuadro de texto con el código html, el cual puede copiar y pegar para incrustar la simulación. Las simulaciones Flash se pueden incrustar o enlazar directamente mediante una imagen para ejecutar la simulación. Las simulaciones Java solo se pueden enlazar mediante una imagen para ejecutar la simulación.
Algunos LMS pueden requerir que use "https://" para el URL de la simulación.

¿Tienen una aplicación?

Ofrecemos na aplicación para iPad que permite el uso offline de todas las simulaciones HTML5. Encuéntrela en la Tienda de Aplicaciones aquí. Nuestra aplicación cuesta $0,99 y corresponde a financiación más desarrollo de simulaciones HTML5.


Actualmente co ofrecemos una aplicación para Android.

¿Dónde puedo encontrar el código fuente?

Puede encontrar los códigos fuente de PhET HTML5 y Javascript en La página GitHub de PhET. Los códigos fuente Java, Flash, y Flex de PhET se encuentran en un repositorio de Subversión. Para más información sobre cómo acceder, consulte la página Código Fuente.

Soy un socio corporativo ¿Debo pagar para usar las simulaciones en mi producto?

Todas las simulaciones disponibles en http://phet.colorado.edu son recursos educativos libres que están disponibles bajo la licencia Creative Commons Attribution (CC-BY).

Se otorga permiso para usar, compartir o redistribuir libremente las simulaciones de PhET bajo la licencia CC-BY. Se requiere de la siguiente atribución:

Simulaciones Interactivas de PhET
Universidad de Colorado en Boulder
http://phet.colorado.edu

Si su uso incluye la redistribución de las simulaciones, por favor háganoslo saber llenando este formulario.

Soy un profesor ¿Necesito una licencia para usar una simulación en mi clase?

No necesita una licencia. Las Simulaciones de PhET son recursos educativos abiertos y usted y sus estudiantes los pueden usar de manera gratuita.

Soy in investigador ¿Necesito una licencia para usar las simulaciones de PhET y publicar mi investigación?

No se necesita una licencia para propósitos de investigación. Por favor háganos saber sobre su investigación llenando este formulario.

Tengo una idea para una simulación ¿Dónde puedo enviarla?

Estamos complacidos de aceptar nuevas ideas y sugerencias para mejorar las simulaciones existentes. Envíe sus ideas aquí!

¿Dónde puedo encontrar soluciones a las actividades en el sitio web?

No todas las actividades tienen respuestas. Algunos autores compartirán las respuestas con profesores miembros, intente enviarle un correo al autor de la actividad.

¿Puedo insertar una simulación en una presentación de PowerPoint?

¡Sí! Puede insertar nuestras simulaciones HTML5 en una presentación de PowerPoint usando nuestra aplicación Office Mix

Esta herramienta funciona en: PowerPoint 2013 Service Pack 1 o posterior, PowerPoint 2016 para Mac, PowerPoint 2016 o posterior, PowerPoint Online.
